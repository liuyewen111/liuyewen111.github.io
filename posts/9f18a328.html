托管不支持 Cloudflare 的免费域名（xx.kg、qzz.io 等）到 LuaDNS

发表于2025-06-02|更新于2025-06-03|建站教程
|总字数:537|阅读时长:1分钟|浏览量:

对于支持 Cloudflare 的域名，域名托管和 DNS 管理变得非常方便。

但对于不支持 Cloudflare 的域名，比如一些新出的免费域名（如 xx.kg、qzz.io、dpdns.org 等），我们需要采用传统 DNS 服务进行托管。

本文将介绍如何使用 LuaDNS 托管这些不支持 Cloudflare 的域名，只要你的域名注册商允许修改 NS 记录，就能按此方法操作。

事前准备

  • 一个可修改 NS 记录的免费域名（如 xx.kg, qzz.io）
  • 域名管理平台账号（如 domain.digitalplat.org）
  • 一个 LuaDNS 账号

注意： xx.kg 目前注册需要 KYC 密钥，普通用户可选择 dpdns.org 或 qzz.io。

托管步骤

进入域名管理控制台

打开：

https://dash.domain.digitalplat.org/panel/main?page=%2Fpanel%2Fdomains

选择你要托管的域名，进入其管理页面。

修改 NS 记录

在域名管理页，将：

  • Name Server 1 修改为 ns1.luadns.net
  • Name Server 2 修改为 ns2.luadns.net
  • Name Server 3 修改为 ns3.luadns.net（可选）
  • Name Server 4 修改为 ns4.luadns.net（可选）

注册 LuaDNS 账号

前往注册页面：
https://app.luadns.com/signup

填写邮箱和密码，点击 Register。随后查收邮箱，会收到一封标题为 LuaDNS Support 的确认邮件，点击其中的确认链接完成注册。

登录 LuaDNS

访问：https://app.luadns.com/login
输入邮箱和密码后，点击 Log in 进入管理界面。

添加你的域名

进入添加页面：
https://app.luadns.com/zones/new

  • 输入你的域名（例如：haib.xx.kg）
  • 模板选择：My Template
  • 点击 Save

查看托管状态

访问：https://app.luadns.com/zones

  • 若域名前显示绿色图标，说明已成功托管。

添加 DNS 解析记录

  • 点击对应域名右侧的 Edit
  • 在新页面中点击右下角的 Add New Records
  • 添加 A 记录、CNAME 等 DNS 配置

总结

即使你的域名不支持 Cloudflare，只要能修改 NS 记录，就可以通过 LuaDNS 等第三方服务实现 DNS 托管与解析。
