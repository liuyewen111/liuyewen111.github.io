使用ZoneABC免费域名搭建属于自己的Cloudflare优选域名

发表于2025-06-02|更新于2025-06-03|建站教程
|总字数:725|阅读时长:2分钟|浏览量:

前言

ZoneABC（zabc.net）提供免费的域名服务。每个用户可注册最多 5 个子域名，总计支持 20 条 DNS 解析记录。

注册账号

进入注册页面

  1. 打开官网：https://zabc.net

  2. 点击右上角「登录」→ 再点下方「没有账号立即注册」

  3. 填写以下信息：

    • 昵称
    • 邮箱
    • 密码（需包含大写字母与特殊字符，建议不要使用 #）

  4. 完成人机验证，点击「发送验证码」

  5. 打开邮箱，复制验证码并粘贴返回页面

  6. 再次验证，完成注册

完善个人资料以获取积分

  1. 登录后进入「账户信息」，拉到下面找到「个人资料」

  2. 补充：

    • 姓名
    • 电话号码
    • 所在城市

  3. 保存信息后，系统会赠送 20 个积分

    注意：如果你不完善资料，初始积分为 0，无法注册部分前缀。

积分使用

前缀长度决定是否需要积分：

前缀长度 是否需要积分 所需积分
7位或以上 无需积分 0
6位 需要积分 20
5位 需要积分 20
4位 需要积分 30

例如：

  • abcdefg.zabc.net（7位）：无需积分
  • cloud.zabc.net（5位）：需要 20 积分
  • cdn.zabc.net（3位）：不可注册（最低为4位）

获取积分方法

  • 完善资料：+20 积分
  • 邀请朋友注册：每邀请 1 人，+10 积分

注册域名

  1. 进入后台 → 点击「域名管理」→「注册域名」
  2. 输入你想要的前缀（系统会自动补上 .zabc.net）
  3. 若所选前缀需要积分，系统会提示所需积分
  4. 点击「立即注册」

每个账户最多注册 5 个域名，总共最多添加 20 条 DNS 记录。

配置CF优选域名

添加并配置 DNS 记录

  1. 在已注册域名右侧点击「管理」

  2. 添加记录：

    • 类型：A
    • IP 地址：104.17.239.90（填写CF优选IP）
    • Cloudflare 代理：关闭（等同于CF的小黄云）


3. 保存记录，几分钟后全球生效

CF优选IP：https://www.wetest.vip/page/cloudflare/address_v4.html

关于优选 IP 的填写建议

如果该优选域名是你自己使用的，那么你应根据当前所使用的网络运营商（如中国电信、中国移动、中国联通等）填写对应的优选 IP。

这样可以最大限度地确保你所使用的网络出口能够直连 Cloudflare 的本地优选节点，减少中转跳数，提升稳定性与访问速度。

示例

  • 如果你是中国电信用户，建议使用电信方向的 CF 优选 IP，如 162.159.xxx.xxx
  • 如果你是中国移动用户，建议使用移动方向的 IP，如 104.16.xxx.xxx
  • 如果你是中国联通用户，则填写联通方向的优选 IP

建议使用itdog定期检测 IP 质量，选择最适合自己网络出口的 Cloudflare 节点 IP。
