## 使用 Windows 自带工具 —— diskpart
这是最通用、最稳定的方法，适用于大多数 PE 盘恢复场景。

步骤如下：
1. 插入 PE 盘，打开命令提示符
在搜索栏输入 **cmd**，然后以管理员身份打开命令提示符。

2. 进入磁盘管理工具
在命令提示符中输入以下命令，并回车：
```
diskpart
```
3. 查看所有磁盘
继续输入：
```
list disk
```
你会看到当前连接的所有磁盘列表。根据容量识别你的U盘是哪一个（比如是“磁盘1”）。

4. 选择目标U盘
假设你的U盘是“磁盘1”，输入：
```
select disk 1
```
一定要确认选对了磁盘，否则可能会误删电脑系统盘！

5. 清除所有分区
输入：
```
clean
```
这条命令会清除U盘上的所有分区和引导信息，相当于“恢复出厂设置”。

6. 创建新的主分区
输入：
```
create partition primary
```
7. 格式化分区
你可以根据需要选择文件系统：

格式化为 NTFS：
```
format fs=ntfs quick
```
格式化为 FAT32（兼容性更好）：
```
format fs=fat32 quick
```
使用 quick 参数是“快速格式化”，速度更快，一般推荐使用。

8. 分配驱动器号
输入：
```
assign
```
这样你的U盘就会重新出现在“此电脑”里啦！

9. 退出 diskpart
最后输入：
```
exit
```
完成全部操作！
